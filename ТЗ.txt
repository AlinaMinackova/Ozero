ТЕХНИЧЕСКОЕ ЗАДАНИЕ на разработку сайта обмена товарами

Перечень этапов разработки сайта:
-	Разработка концепции сайта, информационное проектирование, оформление Технического задания
-	Разработка эскиза базового дизайна
-	Разработка внутренней части сайта
-	Создание работающего шаблона сайта, включая полную разработку «визуализации», ссылки

1. СТРУКТУРА И ОПИСАНИЕ САЙТА:
Главная страница (обложка сайта):
•	Содержит графическую часть, основную навигацию, а также контентную область для того, чтобы посетитель сайта с первой страницы мог получить вводную информацию о компании, а также ознакомиться с объявлениями.
•	Контентная область первой страницы делится на разделы:
1.	«Каталог» - перебрасывает нас на все объявления с возможностью               найти нужный товар
2.	«Главная» - перебрасывает на основную страницу сайта
3.	«Корзина» - перенаправляет в корзину пользователя
4.	Внизу страницы отображается облегченная навигационная панель с ссылками, контакты.

 Графическая оболочка внутренних страниц (общая для всех подразделов):

•	Графическая шапка 
•	Внизу страницы отображается облегченная навигационная панель с ссылками, контакты.

Описание контента разделов сайта:
1.	Главная страница
2.	Каталог
3.	Корзина
4.	Формы регистрации/входа
5.	Страница товара
6.	Страница доставки
7.	Добавление, изменение объявлений

Главная страница:
•	О сайте - текст (информация о компании и предоставляемых услугах) + фото – в свободной форме
•	Контакты – текст с контактной информацией, форма для обратной связи
•	Объявления пользователей – ссылки включающие:
	    Имя пользователя
	    Название объявления	
	    Цену
	    Адрес
	    Фото товара

Каталог:
•	Поиск по объявлениям
•	Товары

Регистрация – форма с полями:
•	E-mail
•	Пароль
•	Повторить пароль
•	Имя
•	Фамилия
•	Телефон
•	адрес

Авторизация (для зарегистрированных пользователей)
•	E-mail
•	Пароль

Корзина (таблица с полями):
•	Номер товара в корзине
•	Название товара (с ссылкой на этот товар в БД каталога)
•	Цена
•	Возможность удалить товар из корзины
•	Общая стоимость заказа

«Регистрация» и «Авторизация» являются обязательными пунктами для добавления объявлений в корзину. Если объявление добавляет незарегистрированный пользователь, его перебрасывает на страницу авторизации.
